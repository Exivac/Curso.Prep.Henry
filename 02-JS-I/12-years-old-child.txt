--Variables--
Las variables te dejan identificar valores con un nombre, para que luego puedas usar el nombre de la variable en otro lado y tantas veces quieras en lugar de escribir el valor cada vez. Tambien puedes cambiarle el valor a una variable para cambiarlo en todas al mismo tiempo sin tener que cambiarlas a mano: "El programa entiende el nombre con el valor".

--Strings--
Los strings sirven como palabras que el programa no tomara como comandos y se pueden poner como valores de una variable para poder almacenar oraciones enteras en una palabra reutilizable. (eso si, si cambias el string en la variable, va a cambiar en todos lados donde lo repitas)

--Funciones (argumentos, `return`)--
Las funciones trabajan como las variables, pero en lugar de almacenar informacion, hacen operaciones sobre informacion que les das (llamada "Argumentos). Las funciones te dan el resultado a travez del comando "return" que funciona como el valor en las variables: "Esta funcion con estos argumentos significa 'esto' (lo que se obtiene con 'return')"


-- Declaraciones `if` --
Declarar "if" le dice al programa que solo haga algo si la condicion que le das se cumple, para lo cual usa los valores booleanos que explicare despues. Si la condicion se cumple, entonces el "if" trabaja y luego se sigue con el resto del codigo. Si la condicion no se cumple, el "if" no trabaja y se continua con el resto del codigo.

-- Valores booleanos (`true`, `false`) --
Los valores booleanos sirven para saber si algo se cumple o no. "true" significa que las condiciones se cumplen y "false" significa que no.
La declaracion "if" usa los booleanos para saber si hacer algo o no.
Tambien puedes usar "return true" o "return false" para hacer que una funcion devuelva un valor booleano y se pueda usar como un "if"
